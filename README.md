# 🛒 Retail Customer Shopping Behavior Analytics Dashboard

An end-to-end **Data Analytics** project that analyzes retail customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI**. The project demonstrates the complete analytics pipeline—from data cleaning and exploratory data analysis (EDA) to database management, SQL querying, and interactive dashboard visualization.

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- PostgreSQL Database Integration
- SQL-Based Data Analysis
- Interactive Power BI Dashboard
- Business KPI & Customer Insights

## 🛠️ Tech Stack

- Python
- Pandas & NumPy
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

## 📊 Dashboard Insights

- Customer Demographics
- Sales Performance
- Product Category Analysis
- Payment Method Trends
- Customer Satisfaction
- Interactive KPI Dashboard
